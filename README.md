## 🎨 设计风格

本工具现在提供三种设计风格供您选择：

### 1. 原始玻璃拟态（默认）
- 半透明玻璃效果
- 蓝色渐变点缀
- 圆润边角
- 柔和阴影
- 适合日常办公使用

### 2. 现代清爽风格（新增 ⭐）
- 采用 oklch 颜色空间，色彩更准确
- 高对比度设计，阅读体验更佳
- 清晰的视觉层次
- 优化的间距和排版
- 适合长时间使用

### 3. 极简单色
- Minimalist Monochrome 设计系统
- 纯黑白配色
- 锐利直角（零圆角）
- 中文优化字体（思源宋体 + 思源黑体）
- 编辑风格，适合专业展示

**切换方法**：
1. 打开 `style-preview.html` 实时预览和对比三种风格
2. 或直接修改 `index.html` 中的样式表引用：
   - 原始风格：`<link rel="stylesheet" href="./styles.css"/>`
   - 现代风格：`<link rel="stylesheet" href="./styles-modern.css"/>`
   - 极简风格：`<link rel="stylesheet" href="./styles-monochrome.css"/>`

**设计对比**：查看 [DESIGN-COMPARISON.md](DESIGN-COMPARISON.md) 了解风格的详细对比

## 功能特点

- 上传 Excel 文件（.xlsx 格式）
- 批量选择和提取指定字段列
- 支持重名列的智能处理
- 拖拽排序已选择的列
- 导出处理后的 Excel 文件
- 自动生成数据处理说明
- 历史配置快速应用

## 快速开始

### 1. 准备库文件

本工具依赖 `xlsx-js-style` 库。系统会自动尝试从多个CDN加载，但建议下载到本地以确保稳定性。

**下载步骤：**

1. 访问以下任一链接下载库文件：
   - https://cdn.jsdelivr.net/npm/xlsx-js-style@1.2.0/dist/xlsx.bundle.js
   - https://unpkg.com/xlsx-js-style@1.2.0/dist/xlsx.bundle.js

2. 将下载的文件保存到 `libs` 目录，文件名为 `xlsx-js-style.bundle.js`

3. 确保文件路径为：`libs/xlsx-js-style.bundle.js`

### 2. 打开工具

直接在浏览器中打开 `index.html` 文件即可使用。

### 3. 使用流程

1. **上传文件**：点击"选择文件"上传 .xlsx 格式的 Excel 文件
2. **选择字段**：
   - 在输入框中输入列名（支持逗号、多个空格分隔）
   - 或从"原始表头"列表中点击添加
   - 勾选"重名列仅添加第一次出现的"可控制重名列的添加行为
3. **调整顺序**：拖拽"已选择列"中的项目调整顺序
4. **导出文件**：点击"导出处理后的文件"按钮

## 主要功能

### 批量输入列名
支持多种分隔符：
- 逗号：`姓名, 电话, 地址`
- 多个空格：`姓名   电话   地址`
- 混合使用

### 重名列处理
- 默认：只提取第一次出现的列（推荐）
- 取消勾选：自动提取所有重名列

### 智能提示
- 输入时显示匹配的列名候选
- 支持多选和批量添加
- 键盘快捷操作（上下键、Enter、Shift+Enter）

### 导出增强
- 重名列自动添加序号标识（如：姓名(1)、姓名(2)）
- 重名列使用黄色背景标识
- 所有单元格添加边框
- 自动生成"说明"工作表，包含：
  - 基本信息和数据统计
  - 重名列分析
  - 提取的列清单
  - 未提取的字段列表
  - 跳过项详情

### 历史配置
- 自动保存常用的列配置
- 快速应用历史配置
- 支持清空历史记录

## 故障排除

如果遇到"Excel 解析库加载失败"的提示，请参考 `TROUBLESHOOTING.md` 文件。

常见问题：
1. 库文件加载失败 → 手动下载到本地
2. 文件无法解析 → 检查文件格式（必须是 .xlsx）
3. 页面显示异常 → 清除浏览器缓存
